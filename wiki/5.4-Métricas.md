# Métricas de Pruebas

Este documento detalla el conjunto de métricas que se recogerán durante el transcurso de ejecución de las pruebas correspondientes a este hito, enfocadas en **Pruebas Unitarias**

### 1. Métricas de Cobertura y Diseño

* **Cobertura de código (Code Coverage):** Target $\ge 85\%$  
  *(Porcentaje de código fuente cubierto por las Pruebas Unitarias verificado en el CI de GitHub Actions).*

* **Número de casos de prueba por requisito:** *(Cantidad de escenarios diseñados para validar cada funcionalidad).*

* **Cantidad de líneas de código:** *(Líneas de código efectuadas exclusivamente para la implementación de los casos de prueba unitarios).*

### 2. Métricas de Ejecución y Calidad

* **Número de casos de prueba ejecutados:** *(Total de pruebas unitarias corridas en el entorno controlado).*

* **Tasa de éxito de pruebas:** 

$$\text{Tasa de Éxito} = \left( \frac{\text{Casos de prueba exitosos}}{\text{Total de casos de prueba ejecutados}} \right) \times 100\%$$
